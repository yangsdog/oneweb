## 页面功能规划

### 1. index.html - 主页
**核心功能**:
- Hero区域：展示公司形象和核心价值
- 产品推荐：精选莲藕产品展示
- 公司亮点：种植规模、技术优势、品质保证
- 新闻动态：最新公司动态展示

**视觉设计**:
- 全屏Hero背景图配合渐变遮罩
- 动态文字效果和颜色循环
- 产品卡片悬停3D效果
- 滚动触发的元素动画

### 2. products.html - 产品中心
**核心功能**:
- 产品筛选系统：按类别、价格、规格筛选
- 产品网格展示：图片、名称、价格、规格
- 产品详情模态框：详细信息、多张图片、购买选项
- 购物车功能：添加、删除、修改数量
- 在线订购：模拟购买流程

**交互设计**:
- 左侧筛选面板（桌面端）/ 顶部筛选栏（移动端）
- 产品卡片悬停效果和快速操作
- 购物车侧边栏滑出动画
- 购买流程的模态框引导

### 3. about.html - 关于我们
**核心功能**:
- 公司简介：历史、规模、理念
- 种藕基地展示：虚拟参观、实景图片
- 种植技术：生长周期、技术优势
- 团队介绍：核心团队成员
- 联系方式：地址、电话、地图

**内容展示**:
- 基地地图交互展示
- 时间轴形式的生长周期
- 团队成员卡片展示
- 联系表单功能

### 4. news.html - 新闻资讯
**核心功能**:
- 新闻分类：公司动态、行业资讯、技术文章
- 文章列表：标题、摘要、时间、阅读量
- 文章详情：富文本内容、图片、分享
- 搜索功能：关键词搜索
- 相关推荐：相关文章推荐

**用户体验**:
- 分类标签切换
- 文章卡片悬停效果
- 搜索结果的即时显示
- 社交分享功能

## 技术实现

### 前端技术栈
- **HTML5**: 语义化标签结构
- **CSS3**: Tailwind CSS + 自定义样式
- **JavaScript**: 原生JS + 第三方库
- **响应式设计**: 移动端优先设计

### 核心库使用
- **Anime.js**: 页面动画和交互效果
- **Splide.js**: 图片轮播和滑动效果
- **ECharts.js**: 数据可视化图表
- **p5.js**: 背景粒子效果
- **Matter.js**: 物理动画效果
- **Pixi.js**: 高性能图像处理
- **Shader-park**: 背景着色器效果

### 交互功能
- 产品筛选和搜索
- 购物车管理
- 模态框和弹窗
- 表单验证和提交
- 页面滚动动画
- 响应式导航菜单
